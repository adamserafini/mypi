# Just My Pi Things

A few tools I always need.

## Installation

You can install these prompt templates and skills directly into your Pi agent configuration from this GitHub repository.

### As a Global Pi Package

Run the following command in your terminal:

```bash
pi install git:github.com/adamserafini/mypi
```

### For a Specific Project Only

If you only want to use these templates in a specific project's settings, navigate to that project and use the `-l` flag:

```bash
pi install -l git:github.com/adamserafini/mypi
```
